This project is an **AI-powered web-based system** designed to summarize and analyze **Sinhala legal documents** in order to improve legal accessibility for non-expert Sri Lankan citizens.

Since Sinhala is a **low-resource language** in Natural Language Processing (NLP), there are very limited tools available for legal document summarization in the Sri Lankan context.
